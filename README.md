# 豆豆数学探险队 v6.0

面向豆豆“三升四”阶段的可安装数学学习应用。它整合了现有暑假计划的节奏、学校数学作业的阶段背景，以及 55 道真实数学思维题。

## 使用

```bash
cd "豆豆数学探险队_v5.0"
npm run build
python3 -m http.server 8092 -d public
```

在同一 Wi-Fi 下，手机访问电脑的局域网地址加 `:8092`；首次打开后可从浏览器菜单“添加到主屏幕”。安装后，题库、练习记录、复盘计划和页面资源可离线使用。

## 学习方式

- 首页每天最多 3 关：一题真实暑假思维题 + 复习/新题。
- “暑假思维营”取自 `豆豆暑假学习计划_v1.0/math_bank.json` 的 55 题，并在构建时复制到应用中。
- 选择题即时判定；开放奥数题先写思路，再看分步讲解后自评掌握，答错自动进入间隔复盘。
- 记录仅保存在本机浏览器，切换年级和昵称不会丢失。

## 发布

`public/` 是可直接部署到 Cloudflare Pages 的静态目录；运行 `npm run build` 后会同步 HTML、脚本、样式、PWA 资源和题库。
